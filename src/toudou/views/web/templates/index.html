How does it work?

Add a task

Simply fill in the "Title" and "End date" fields and click "Add".

Mark a task as completed

You need to click "Complete" in the "Done" column.

You can also mark a task as in progress by clicking "In progress".

Delete a task

Click "Delete" in the "Actions" column of the task you want to delete.

Delete multiple tasks

You can select the tasks you want to delete and click "Delete" located below the task list.

Export tasks

Select the tasks you want to export and click "Download" located below the task list.

Import tasks

Choose a CSV file containing the tasks you want to import and click "Upload" located below the task list.
